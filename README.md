# Monkadex

Monkadex is a website created by Simon Aarseth (Simonkamon11) as a tool for all things Pokémon, the main page being a common Pokédex. 

## Features
- Pokédex
- Pokémon locations
- Move searcher (under construction)
- Shiny tools
